# The Astryx Dominion

This Obsidian vault contains the comprehensive lore, characters, quests, and game systems for the Dungeons & Dragons world called "The Astryx Dominion."

## Overview

The Astryx Dominion is a science fiction Dungeons & Dragons campaign set in a vast galactic universe. It features custom species, factions, technologies, and adventures blending traditional fantasy elements with space opera themes.

## Vault Structure

- **Categories/**: Organized notes on various campaign elements like species, items, quests, and more.
- **DM STUFF/**: Dungeon Master resources including homebrew systems, player characters, and quest details.
- **Factions, Star-nations, Megacorps, and More/**: Information on major galactic powers and organizations.
- **Galactic Species/**: Detailed entries on playable and NPC species adapted for the campaign.
- **Galactic Tech/**: Technological systems, operating systems, and other sci-fi elements.

## Getting Started

This vault is designed for use with Obsidian. To explore the campaign lore:

1. Open this folder in Obsidian.
2. Start with the main category files in `Categories/` for an overview.
3. Navigate to specific sections based on your needs (e.g., species, quests, factions).

## Plugins and Features

This vault utilizes several Obsidian plugins:
- Dataview for dynamic note querying
- Calendarium for campaign timeline management
- Smart Connections for AI-assisted note linking
